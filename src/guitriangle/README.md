2D Gui triangle rendering class

## Installation
```
npm install @quenty/guitriangle --save
```

## Usage
Usage is designed to be simple.

### `GuiTriangle.new(parent)`

### `GuiTriangle:SetParent(parent)`

### `GuiTriangle:Show()`

### `GuiTriangle:Set(a, b, c)`

### `GuiTriangle:Hide()`

### `GuiTriangle:SetA(a)`

### `GuiTriangle:SetB(b)`

### `GuiTriangle:SetC(c)`

### `GuiTriangle:UpdateRender()`

### `GuiTriangle:Destroy()`
