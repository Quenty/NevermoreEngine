```

```

## Usage
Usage is designed to be simple.

### `ActionManager.new()`

### `ActionManager:StopCurrentAction()`

### `ActionManager:ActivateAction(name, ...)`

### `ActionManager:GetAction(name)`

### `ActionManager:GetActions()`

### `ActionManager:AddAction(action)`
